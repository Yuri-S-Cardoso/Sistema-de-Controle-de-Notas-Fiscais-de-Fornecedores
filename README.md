Sistema de Controle de Notas Fiscais de Fornecedores

Sistema web desenvolvido em Django para gerenciamento e acompanhamento de notas fiscais de fornecedores, permitindo
o controle completo desde o cadastro do fornecedor até o envio das notas ao setor financeiro.

Funcionalidades

* Cadastro de fornecedores por CNPJ
* Definição de notas recorrentes por fornecedor
* Registro mensal de notas fiscais recebidas
* Controle de datas de emissão, entrega e vencimento
* Acompanhamento do envio para cadastro e financeiro
* Edição de registros de notas fiscais
* Filtros por período (emissão e vencimento)
* Relatório mensal consolidado com total de valores por fornecedor
* Paginação de registros para melhor navegação

Tecnologias Utilizadas

* Python
* Django
* HTML / CSS
* SQLite / Banco relacional
* Django ORM

Estrutura do Sistema

O fluxo de funcionamento segue três níveis principais:

1. Fornecedor → cadastro da empresa
2. Tipo de Nota → definição de cobranças recorrentes
3. Notas Mensais → registro das notas fiscais efetivamente recebidas

Aplicação

Indicado para ambientes corporativos que necessitam controlar o recebimento e processamento de notas fiscais
de fornecedores, especialmente em rotinas administrativas e financeiras.
